# AutoML Demo

デモアプリケーションのフロントエンド。

## 動作環境

Viual Studio Code

https://code.visualstudio.com/

Angular CLI
version 8.1.1

https://github.com/angular/angular-cli

## セットアップ

`next19demo/frontend` のディレクトリで

`npm install`

## ローカル実行

`npm start`

ローカルの URL
http://localhost:4200/

## ビルド

ローカル動作用

`ng build`

本番環境用

`ng build --prod`

## テスト

`ng test`

[Karma](https://karma-runner.github.io)が起動します。

## e2e テスト

`ng e2e`

[Protractor](http://www.protractortest.org/)が起動します。
